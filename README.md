# 📊 NHANES Multivariate Analysis

An end-to-end multivariate statistical analysis pipeline on the **NHANES 2017–2018 dataset**, implemented using **Python and R**.  
This project demonstrates practical application of advanced statistical techniques for real-world health data analysis.

---

## 🚀 Project Overview

This project focuses on applying multivariate analysis techniques to uncover hidden patterns, relationships, and structures within health-related data.

Key objectives include:
- Data preprocessing and cleaning  
- Exploratory Data Analysis (EDA)  
- Dimensionality reduction  
- Modeling relationships between variables  
- Interpreting statistical results in a real-world context  

---

## 🧠 Techniques Used

The following multivariate methods are implemented:

- 📌 **Exploratory Data Analysis (EDA)**  
- 📉 **Principal Component Analysis (PCA)**  
- 🔍 **Factor Analysis (FA)**  
- 🧮 **Discriminant Analysis (DA)**  
- 🔗 **Canonical Correlation Analysis (CCA)**  
- 🏗️ **Structural Equation Modeling (SEM)**  

---

## 🗂️ Dataset

- Source: **NHANES (National Health and Nutrition Examination Survey) 2017–2018**
- Type: Real-world health and nutrition data
- Includes variables such as:
  - Demographics  
  - Body measurements  
  - Laboratory results  
  - Health indicators  

---

## ⚙️ Tech Stack

**Languages & Tools:**
- Python (Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn)  
- R (lavaan, psych, stats)  
- Jupyter Notebook / Google Colab  

---

## 🔄 Workflow

1. **Data Collection & Integration**  
   - Load and merge multiple NHANES datasets  

2. **Data Preprocessing**  
   - Handle missing values  
   - Remove/adjust outliers  
   - Normalize/scale variables  

3. **Exploratory Data Analysis (EDA)**  
   - Distribution analysis  
   - Correlation analysis  
   - Visualization  

4. **Dimensionality Reduction**  
   - PCA for variance explanation  
   - FA for latent variable identification  

5. **Multivariate Modeling**  
   - DA for classification  
   - CCA for inter-variable relationships  
   - SEM for structural modeling  

6. **Interpretation & Insights**  
   - Evaluate assumptions  
   - Interpret model outputs  
   - Identify meaningful patterns
